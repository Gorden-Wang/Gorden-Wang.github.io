JavaScript

  • CSS3 media Query

    跨终端的Web设计不紧紧是Media Query，但是Media Query是其中最重要的展现形式。除此之外还有多站点，多模板等多种实现方式。Media Query在展现层会有些优势，但是也会有些劣势，比如DOM冗余和JS冗余等。现在讨论的是如何去设计Media Query

    当然，现在的趋势是移动优先，但是也不建议以移动为主去写CSS，因为移动端的DOM元素相比PC会少很多，业务逻辑和交互也有很大差别，建议以PC为主，再单独做Mobile的样式。

    我们推荐BootStrap的Media Query，去适配Large Screen （>1200px）,Middle Screen(992-1200px),Tablets(768~992),Small Screen(<768px)。

                        @media (max-width: $screen-sm-min){}

                    @media (min-width: $screen-md-min){}

                    @media (min-width: $screen-lg-min)
